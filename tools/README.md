# BOSS直聘 自动筛选打招呼

> 在「推荐牛人」页面，按你设定的条件（公司、职位、是否应届）自动筛选候选人并打招呼。

## 安装（3 步）

1. 浏览器安装 [Tampermonkey 扩展](https://www.tampermonkey.net/)
2. 点击 Tampermonkey 图标 → 「添加新脚本」
3. 把 `boss_auto_greet.user.js` 的全部内容粘贴进去，保存

## 使用

1. 打开 BOSS直聘「推荐牛人」页面（`zhipin.com/web/frame/recommend?...`）
2. 页面右上角会出现绿色的「🤖 自动打招呼」面板
3. 检查/修改你的筛选条件（公司白名单、职位关键词等）
4. 点 **▶ 开始**

面板会实时显示：
- 当前正在处理谁（黄色高亮）
- 每个候选人的筛选结果（✅通过 / ❌淘汰 + 原因）
- 每次 API 请求的批次统计（拿了多少人、过滤了多少）
- 打招呼的结果（🤝成功 / ⚠️失败）

点 **📋 日志** 可以查看完整的筛选记录，支持按「通过/淘汰/已打招呼」筛选。

## 注意事项

- **不要贪多**：每日建议 ≤50 人，新号首周 ≤30
- **不要太快**：打招呼间隔建议 8~15 秒
- **遇到验证码立即停止**，等半小时以上再用
- 脚本获取的候选人和页面上看到的可能不同，这是正常的（两边是独立的请求）
